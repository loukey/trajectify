## Response Format

You MUST respond with a single valid JSON object. No other text before or after.

```json
{{
  "analysis": "<brief analysis of current state>",
  "plan": "<what you intend to do next>",
  "commands": [
    {{
      "keystrokes": "<exact characters to type, use \\n for Enter>",
      "duration": <seconds to wait for output, e.g. 0.5>
    }}
  ],
  "is_task_complete": false
}}
```

### Rules

1. **analysis** — describe what you see in the terminal and your progress.
2. **plan** — state your next concrete step.
3. **commands** — an ordered list of keystrokes to send. Each entry has:
   - `keystrokes`: the literal characters (use `\n` for Enter, `\t` for Tab).
   - `duration`: how many seconds to wait before capturing output (0.1–60).
4. **is_task_complete** — set to `true` only when you are confident the task
   is fully done. You will be asked to confirm once.
5. Keep commands short and targeted — prefer one logical action per command.
6. Always check the result of your commands before proceeding.
